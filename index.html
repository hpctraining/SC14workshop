Workshop on Best Practices for HPC Training @SC14

Date: Friday, 21 November, 2014

Deadline for abstract submission: August 31st, 2014

Workshop Format

The workshop will be a mix of short (10-12 minute) contributed presentations, a panel discussion, and an open discussion in which we will develop a workshop report. Presentations and materials will be submitted to github. Please let Fernanda know your github account to be added as a contributor. This github space will be used on future collaboration events.
